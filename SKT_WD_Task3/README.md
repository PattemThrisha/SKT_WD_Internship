# SKT WD Task 3 - Tic Tac Toe Game

## Description
This project is a Tic Tac Toe game developed as part of the Web Development Internship at SkillCraft Technology. It is a simple two-player game where players take turns marking spaces, and the system automatically detects the winner.

## Tech Stack
HTML  
CSS  
JavaScript  

## Features
Two-player gameplay  
Winner detection logic  
Draw detection  
Game reset functionality  
Interactive user interface  

## How to Run
1. Download or clone the repository  
2. Open the SKT_WD_Task3 folder  
3. Open index.html in any web browser  

## Learning Outcome
Learned game logic implementation, event handling, and DOM manipulation using JavaScript.
