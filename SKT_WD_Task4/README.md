# SKT WD Task 4 - To-Do List Web Application

## Description
This project is a simple To-Do List web application developed as part of the Web Development Internship at SkillCraft Technology. It allows users to add and manage daily tasks efficiently.

## Tech Stack
HTML  
CSS  
JavaScript  

## Features
Add new tasks  
Delete tasks  
Simple and clean user interface  
Real-time updates  

## How to Run
1. Download or clone the repository  
2. Open the SKT_WD_Task4 folder  
3. Open index.html in any web browser  

## Learning Outcome
Learned DOM manipulation, event handling, and building interactive web applications using JavaScript.
