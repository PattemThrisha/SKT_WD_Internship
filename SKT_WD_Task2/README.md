# SKT WD Task 2 - Calculator Web Application

## Description
This project is a simple calculator web application developed as part of the Web Development Internship at SkillCraft Technology. It performs basic arithmetic operations with an interactive user interface.

## Tech Stack
HTML  
CSS  
JavaScript  

## Features
Performs addition, subtraction, multiplication, and division  
User-friendly interface  
Real-time input and output display  
Responsive design  

## How to Run
1. Download or clone the repository  
2. Open the SKT_WD_Task2 folder  
3. Open index.html in any web browser  

## Learning Outcome
Learned JavaScript fundamentals, DOM manipulation, and how to build interactive web applications.
