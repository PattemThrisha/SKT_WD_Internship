# SKT WD Task 1 - Responsive Landing Page

## Description
This project is a responsive landing page created as part of the Web Development Internship at SkillCraft Technology. It is designed to adapt to different screen sizes and provide a clean and user-friendly interface.

## Tech Stack
HTML  
CSS  

## Features
Responsive design for all devices  
Simple and clean user interface  
Navigation bar with sections  
Modern layout structure  

## How to Run
1. Download or clone the repository  
2. Open the Task1 folder  
3. Open index.html in any web browser  

## Learning Outcome
Learned how to build a responsive web page using HTML and CSS, including layout design and styling techniques.
